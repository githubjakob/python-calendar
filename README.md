# python-calendar

This software shall allow users to book an appointment.
Choosing from a list of available time slots.


## Setup

Create venv

```
python3 -m venv .venv
```

Activate venv

```
.venv\Scripts\activate.bat

source .venv/bin/activate
```

Install dependencies

```
pip install -r requirements.txt
```

Leave venv

```
deactivate
```

## Tests

```
python -m pytest
```
